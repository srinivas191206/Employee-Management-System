# Employee Management System
A simple web-based Employee Management System built using **Flask** (Python) and **SQLite**.  
This project was developed as part of my **Python Full Stack program** for self-development at *Aditya Institute of Technology and Management*.

## Features
- Add new employees with name, email, and department
- View all employees in a structured table
- Delete employees by ID
- Data stored persistently in SQLite database

## Tech Stack
- Backend : Python (Flask framework)
- Database: SQLite
- Frontend: HTML templates (Jinja2 with Flask)

## Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/employee-management.git
   cd employee-management
2. Create a virtual environment and install dependencies:
   python -m venv env
   source env/bin/activate   # On Linux/Mac
   env\Scripts\activate      # On Windows
   pip install flask
3. Install dependencies:
   pip install flask
4. Run the application:
   python app.py
5. Open your browser and go to:
   http://127.0.0.1:5000/
